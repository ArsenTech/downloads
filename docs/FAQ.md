# Frequently Asked Questions

## What is this?
This is a curated downloads collection by ArsenTech.

It includes selected software such as Windows images, programs, drivers, and other useful files.  
This project is not meant to replace other download sites, its purpose is to provide a simple and organized way to access useful files easily.

No advertisements are used, and all downloads are free.

---
## I'm a copyright owner and want files removed
Please contact me here:  
➡️ https://arsentech.github.io/#contact  

Your request will be reviewed as soon as possible.

---
## Are these files cracked or modified?
No, all files are original copies.

We do not provide cracks or bypass tools.  
If activation is required, users must use their own legitimate licenses.

---
## Downloads are slow — why?
Download speed depends on:
- your internet provider
- your location
- your connection stability
- network limits

The site itself does not throttle speeds.

---
## Who made the coding wallpapers?
All coding-themed wallpapers are created by ArsenTech.

---
## Do you provide malware downloads?
Nope, Malware is never distributed here.  
Sharing malicious software can harm systems and people, so it is intentionally excluded.

---
## Why were Windows Vista–11 files removed?
Due to storage limitations, those versions are no longer hosted directly.

Instead, official sources or trusted archives are provided so users can download authentic versions from reliable providers.
Always verify file integrity after downloading.

---
## Why are file sizes and dates no longer shown?
Because of storage changes (Firebase → GitHub) and GitHub API limits, file size and creation dates can no longer be displayed automatically.

You can still check file size after downloading. If you need this info before downloading, feel free to contact me.
